# EDA-Project---SQL

Project Overview

This project focuses on performing deep Exploratory Data Analysis (EDA) on a retail sales dataset using PostgreSQL/MySQL. The goal was to extract meaningful business insights regarding profitability, regional performance, and seasonal growth trends to support data-driven decision-making.

Technical Skills Demonstrated
Advanced Querying: Utilization of JOINs, GROUP BY, and HAVING clauses for data aggregation.

Window Functions: Implementation of DENSE_RANK() and ROW_NUMBER() to rank regional products and identify top performers.

Common Table Expressions (CTEs): Structured complex logic using multiple CTEs to improve readability and performance for multi-step analyses.

Temporal Analysis: Executed Year-over-Year (YoY) and Month-over-Month (MoM) growth comparisons using CASE statements and date extraction functions.

Data Transformation: Performed schema modifications (ALTER TABLE) and data updates to optimize the dataset for time-series analysis.

Key Business Questions Addressed

Profitability: Identified the top 10 highest profit-generating products and analyzed profit growth by sub-category.

Sales Dynamics: Calculated Average Order Value (AOV) and identified sales peaks across different product categories.

Regional Performance: Determined the top 3 highest-selling products in each geographical region based on quantity sold.

Growth Metrics: Compared monthly sales performance between 2022 and 2023 to identify growth patterns and seasonal variances.
